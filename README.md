## 📌 Overview

**ChatBoat AI Assistant** is a smart conversational chatbot that leverages **Groq’s ultra-fast LLM (LLaMA 3.3)** integrated with **LangChain** to deliver real-time, intelligent responses.

🔍 Designed for:
- AI exploration  
- Learning LLM integration  
- Building scalable chatbot systems  

---

## 🧠 Features

- 🤖 Real-time AI chat interface  
- ⚡ Lightning-fast responses using Groq  
- 🎛️ Adjustable model parameters (temperature)  
- 💬 Context-aware conversations  
- 🔐 Secure API key handling  
- 🎨 Clean and interactive Streamlit UI  
- 🧩 Modular and extensible architecture

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/neurochat-ai.git
cd neurochat-ai
```
### 2️⃣ Setup virtual environment
```bash
python -m venv venv
```
### 3️⃣ Activate Environment
```bash
venv\Scripts\activate
```
or mac\linux
```bash
source venv/bin/activate
```
### 4️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```
---
## 🔑 API Setup
Get your API key from 👉 https://console.groq.com/
```bash
export GROQ_API_KEY="your_api_key"
```
### ▶️ Run Application
``` bash
streamlit run app.py
```

## 🌐 Deployment
